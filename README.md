# Python-Book-Library

Desktop-приложение локальной библиотеки книг на Python.

Описание:

Приложение имеет внутреннюю базу данных, в которой хранятся данные о книгах, добавленных в неё. 
Имеется сортировка по названию, автору, дате публикации, жанру, серии. Можно добавить книгу в базу, удалить, редактировать информацию о ней, а также сканировать всю директорию на наличие книг нужный форматов [.epub, .fb2, .pdf]. Есть окно Preview в котором можно добавить обложку книги, пишется информация о книге и краткое описание книге в окне слева. Существует поиск книг в строке ввода.

Использованные библиотеки и языки:
-Python
-Tkinter
-Sqlite
-EbookLib

Возможности: 

- Добавление книг в базу данных;
- Удаление книг из базы;
- Редактирование информации о книге;
- Поиск книг по строке ввода;
- Сортировка по названию, автору, дате публикации, жанру и серии;
- Сканирование директории с книгами;
- Отображение preview-окна с обложкой, информацией о книге и кратким описанием.

Установка и запуск:
  1.Перейти в директорию проекта и создать виртуальное окружение - в cmd команда "python -m venv .venv".
  2. Активировать его - ".venv\Scripts\activate"
  3. Установить зависимости - "pip install -r requirements.txt"
  4.Запустить - "python app.py"

примечание:
  Файл requirements.txt содержит список внешних Python-библиотек, необходимых для запуска проекта.
